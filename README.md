# 🚀 Deadline Guardian

### AI-Powered Last-Minute Life Saver

Deadline Guardian is an intelligent AI productivity assistant that helps users avoid missed deadlines by analyzing tasks, predicting risks, generating execution plans, and providing emergency recovery strategies using Google's Gemini AI.

🌐 **Live Demo:** https://deadline-guardian-67sv.onrender.com/

📂 **GitHub Repository:** https://github.com/Sai-Sri-Krishna/Deadline_Guardian

---

## 🎯 Problem Statement

Students, professionals, and teams frequently miss important deadlines due to poor planning, underestimating effort, and ineffective prioritization.

Traditional reminder applications only notify users but do not actively help them complete tasks.

Deadline Guardian transforms passive reminders into an intelligent AI-powered execution system that:

* Analyzes task complexity
* Predicts completion risks
* Generates action plans
* Provides recovery strategies
* Tracks productivity progress

## 🚀 Local Setup

### Clone Repository

```bash
git clone https://github.com/Sai-Sri-Krishna/Deadline_Guardian.git
cd Deadline_Guardian
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Start Application

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:3000
```
